# Task Manager Web Application

This is a simple full-stack Task Manager web application built to manage daily tasks easily.  
Users can add, view, update, and delete tasks through a clean and easy-to-use interface.

The main purpose of this project is to understand how frontend and backend work together and to practice basic CRUD operations.

---

## Features

- Create new tasks
- View the list of tasks
- Edit existing tasks
- Delete tasks when they are no longer needed
- Simple and clean user interface
- Clear separation between frontend and backend code

---

##Tech Stack

### Frontend
- HTML for structure
- CSS for styling
- JavaScript for functionality

### Backend
- Node.js
- Express.js
